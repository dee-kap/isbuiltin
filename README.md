# isbuiltin 

Check if the module is a built-in node module.

## Install

```
npm install isbuiltin --save
```

## Usage

```
var isBuiltIn = require('isbuiltin);

isBuiltIn('console');
=> true

isBuiltIn('woosh');
=> false

```

## Run Tests

```
npm install && npm test
```
