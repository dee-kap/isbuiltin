# isbuiltin 

Tells you if the string you pass in is a built-in node module

## Install

```
npm install isbuiltin --save
```

## Usage

```
var isBuiltIn = require('isbuiltin);

isBuiltIn('console');
=> true

isBuiltIn('woosh');
=> false

```
